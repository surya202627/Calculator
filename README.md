# Console-Based Calculator in Java

A simple and beginner-friendly **Console-Based Calculator** built using **Java**. This application allows users to perform basic arithmetic operations through a menu-driven interface. It demonstrates fundamental Java programming concepts such as methods, loops, conditional statements, and user input handling.

## Features

*  Addition
*  Subtraction
*  Multiplication
*  Division
*  Menu-driven interface
*  Continuous calculations until the user exits
*  Division-by-zero handling
*  Clean and user-friendly console output

## 🛠️ Technologies Used

* Java
* Scanner Class
* Methods
* Switch Statement
* Do-While Loop
* Exception and Input Validation (Basic)

##  How to Run

1. Clone this repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd Calculator
```

3. Compile the Java program:

```bash
javac Calculator.java
```

4. Run the application:

```bash
java Calculator
```

5. Follow the on-screen menu to perform calculations.
